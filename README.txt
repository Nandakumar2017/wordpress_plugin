== Description ==

Email chat Plugin works on any wordpress site build specifically for SEED website used to send email notifications to SEED staffs.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('email_chat_hook'); ?>` in your templates
